# 🧮 JavaFX Calculator

A professional calculator built using Java and JavaFX.

## ✨ Features
- Basic arithmetic operations (+, -, *, /)
- Modern dark UI
- Keyboard support
- Responsive button layout

## 🛠️ Technologies
- Java
- JavaFX

## 🚀 How to Run
1. Compile:
javac --module-path "PATH_TO_FX/lib" --add-modules javafx.controls FXCalculator.java

2. Run:
java --module-path "PATH_TO_FX/lib" --add-modules javafx.controls FXCalculator
